# Moral-IT Deck 改変版（高校情報科向け）

「モラルITデッキ（Japanese version）」を改変したバージョンです。高校情報科の授業で使いやすいよう調整しました。

## 元作品について

このツールは以下の作品を改変した二次的著作物です。

- **作品名**：Moral-IT Deck (Japanese version)  
- **著者**：Yusuke Shikano, Eri Mizumachi, Konomi Higo & Kohei Takeda  
- **© 2023**  
- **元URL**：https://elsi.osaka-u.ac.jp/program_tool/2534

元作品は大阪大学社会技術共創研究センター（ELSIセンター）が公開したものです。[page:42]

## 改変内容

以下の点を変更・追加しました：
- **カードデッキ**：高校情報科のカリキュラムに合わせ、カード内容を敷衍。カード裏面に説明文を追加。
- **進行用スクリプト**：授業時間（50分授業）を想定し、ステップを簡略化。ファシリテーター向けの時間配分メモを追加。アスペクト時間論の観点からテクノロジーを分析する視点を導入。

## 使い方

1. **カードデッキ**：PDFを印刷（L判推奨）。[page:42]  
2. **進行用スクリプト**：ファシリテーターが読みながら進行。  
3. 所要時間：45〜60分（高校授業向け）。

テクノロジーの社会的影響を対話で評価するインパクトアセスメントツール。英国で開発された"Moral IT-Deck"の日本語版を高校情報科向けに改変し、アスペクト時間論の観点を取り入れたワークショップ進行用スクリプトを新規開発。ELSIの評価を通してRRI（責任ある研究・イノベーション）の4次元（先見性・省察性・包摂性・応答可能性）を育成し、テクノロジーがもたらす倫理的課題への対峙を支援する。カードデッキ、進行スクリプトを含む。CC BY-NC-SA 4.0ライセンス。大阪大学ELSIセンター開発。
